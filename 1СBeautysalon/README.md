# Проект: Салон красоты

## Описание проекта
Комплексная автоматизация малого предприятия сферы услуг (салон красоты).

## Реализовано
- Оперативный учёт деятельности предприятия
- Аналитические отчёты
- Подсистема бухгалтерского учёта
- Подсистема расчёта заработной платы
- Загрузка данных из Excel в справочники

## Технологии
- 1С:Предприятие 8.3
- Конфигуратор
- Язык запросов, СКД
- Мобильная платформа 1С
